# craylib_invaders
invaders in c and raylib

# Makefile
### build
```console
make
```
### Run
```console
make run
```
### clean
```console
make clean
```

# keys
game controls

left arrow <kbd><-</kbd> left movement

right arrow <kbd>-></kbd> right movement

up arrow <kbd>^</kbd> shoot

'p key <kbd>P</kbd> pause game

esc <kbd>Esc</kbd> quit game
